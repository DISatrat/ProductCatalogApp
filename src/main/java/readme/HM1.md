# Product Catalog Service

## Описание домашней работы

Реализована система управления каталогом товаров с консольным интерфейсом. Проект демонстрирует применение принципов объектно-ориентированного программирования и многослойной архитектуры.

### Выполненная работа:

- Реализован полный цикл CRUD операций для товаров
- Разработана система аутентификации и авторизации пользователей
- Внедрена ролевая модель с двумя уровнями доступа: USER и ADMIN
- Реализован механизм кэширования поисковых запросов для повышения производительности
- Добавлено ведение журнала аудита всех действий пользователей
- Реализован сбор метрик производительности системы
- Организовано persistent хранение данных между сеансами работы

## Используемый стек технологий

- Java 17
- Maven
- Java Collections Framework
- Java Serialization API
- Java Stream API
- Java NIO для работы с файлами

## Инструкция для запуска

### Предварительные требования:
- Установленная Java 17 или выше
- Установленный Maven 3.6 или выше

### Команды для запуска:

1. Клонирование проекта:
```bash
git clone <repository-url>
cd product-catalog
mvn clean compile
mvn package
java -jar target/product-catalog-1.0.jar
```

### Учетные данные для входа
При первом запуске автоматически создается административная учетная запись:

Логин: ```admin```

Пароль: ```1```
